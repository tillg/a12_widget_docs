Today we are proud to announce and launch the release of our newly redesigned Widgets showcase. Some major improvements you’ll notice are:

- Clearer and more detailed documentation
- API and theming documentation now available inside the showcase
- More relevant example usage of Widgets
- Many class components have been converted to functional components
- Code snippets are now dynamic and generated based on the current state of the showcase
- Improved search results and suggestions
- A better UX navigating between pages and sections of a page
- Inline bug icon buttons to conveniently report bugs

We’d Love to Hear From You

If you’d like to provide high-level feedback on our new showcase, you can do so by sending an email to the [*email*a12-widgets-team](mailto:a12-widgets-team@mgm-tp.com "E-mail").

Otherwise, if you notice issues with a specific showcase please feel free to click one of the bug icon buttons (

*bug\_report*

) in said showcase, and a pre-formatted email with the background info we need to support you (version, browser, device, etc.) will be generated.

*link*Padding Configuration

You can customize the padding of the content in **Content Box** by using the `padding` property. The value of the padding is set to:

- **true** (default): There is a padding to the left, right, and bottom of the Contentbox's content.
- **false**: No padding applied.
- **custom value**: You can also set your custom value for padding. For example: `padding="12px 24px"` or `padding=24`.

*link*Accessibility

To improve Accessibility, the **Content Box** provides some titles based on the locale for:

- **ContentBoxElements.Title**: The title always has the `role = heading`. But there is no default aria-level. You have to pass it in manually by using `ariaLevel` property.
- **Footer**: There is a hidden heading which is placed inside the footer if it contains children. The `headingTitle` property is used to customize the title of the hidden heading. By default, it would be:
  - English: *"Action Section"*
  - German: *"Aktionsbereich"*

  Refers to [Basic Contentbox](../Widgets.Layout.ContentBox#basic/index.md) example.
- **Back button**: The ContentBox provides a navigation back button which is placed inside the heading and before the title. This button has a localized title:
  - English: *"Back"*
  - German: *"Zurück"*

  Refers to [Navigation](../Widgets.Layout.ContentBox#navigationContentbox/index.md) example.

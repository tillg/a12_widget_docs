*link*Helper Classes

Widgets provide a great amount of helper classes (styles), which can be used to change the styling of a specific element/Widget.

*keyboard\_arrow\_right*

Some notes about the usage

*link*Background Color

The **Background Color** Helper Classes can help you to add the **background colors** described in the [Widgets Colors System](../Basics.Theme.Colors/index.md) to the following specific elements/Widgets:

- 1st level Section
- Autocomplete
- Buttons (excluding Toggle Buttons)
- Checkbox / Checkbox Group
- Collapsible Panel
- Icon Picker
- Multiselect
- Radio
- Select
- Text Field
- Text Area
- Text Output
- Time Picker
- Year and Month Selector

The example below shows how these Widgets behave when having a **Background Color** Helper Class.

Warning

*warning*

**NOTE**

Take care when changing the background color as failing to do so may result in poor contrast for:

- The text color of the Widget
- The box shadow of Text Field, Text Area and Select
- The hover color of a Button

Choose the helper class to apply:

defaulth\_blueBGh\_blueDarkBGh\_blueLightBGh\_greenBGh\_greenDarkBGh\_greenLightBGh\_greyBGh\_greyDarkBGh\_greyLightBGh\_orangeBGh\_orangeDarkBGh\_orangeLightBGh\_purpleBGh\_purpleDarkBGh\_purpleLightBGh\_redBGh\_redDarkBGh\_redLightBGh\_yellowBGh\_yellowDarkBGh\_yellowLightBGh\_whiteBGh\_transparentBGh\_blackBG

`default`: `undefined`
